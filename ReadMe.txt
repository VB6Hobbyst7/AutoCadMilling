This is AutoCad plugin that convert excel file with data of existing 
and project elevations of pavement in drawing of milling depth of 
every cross section of road.

This is work version of plugin.

To add it in Acad:
	1. Add Your data in \\AutoCad_milling\AutoCad_milling\bin\Debug\DataInputData.xlsx . Use the format from the file! 
		(Must use exact file name and file format and file location this version of plugin).
	2. Open new drawing ACad or C3D;
	2. Type - Netload;
	3. Navigate to \\AutoCad_milling\AutoCad_milling\bin\Debug;
	4. Choise AutoCad_milling.dll file and load it;
	5. In command line type: MillingDiagramCreator and press Enter;
	6. Prest x2 scroll button of Your mouse OR in command line type "Z" + Enter and "E" + Enter to Zoom Extents.

Soon more will have features.
